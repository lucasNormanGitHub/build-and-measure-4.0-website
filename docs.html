Build and Measure

About

Build and Measure, also known as BAM, is a user-friendly 2D drafting program that comes at no cost, aiming to provide a lightweight experience for users.
Build and Measure is created and maintained by a single developer, and if you find any issue and bugs, please report them and send feedback, it really helps!

Setup

Build And Measure is cross platform and available for:
Windows
MacOS
Linux

Project management

Creating and saving projects

To create a project, press the "create project" button in the left navbar when in the landing page.
This will open the editor with an unsaved project which you can now save by pressing the save button. The first time you press the save button in a new project you will be required to specify a file location but after that the file location will be remembered.

You can also save a project with the "save as" button which write the project to a new file, although the file location won't be stored so the save button will still save in the same file

Opening projects

To open an existing project from a file, press the "open project" button in the left navbar when in the landing page.
This will let you let you select a project in the file explorer which will then be opened in the editor, when you press save the changes will now be written to the opened file

The project editor



Control panel

The control panel is positioned in the top left corner of the editor and has buttons to for example save and export the project.

Home button

The home button takes you back to the start page.

Save button

The save button saves the project. More on saving projects here.

Save As

The save as button lets you choose a file to save the project in. More on saving projects here.

Export button

The export button toggles a dropdown with two options to export to either SVG or PDF.

Views and sheets

A project consists of multiple views and sheets.
Views and sheets are similar in many ways, but one difference is that the sheets allow you to export the a drawing within a title block. A view is where you can draw in 1:1 scale, but to export to a drawing with a title block you need to use a sheet

Drawing objects

To draw an object, select which object to draw in the tool-bar and then press on the canvas to start drawing.
Each object requires a different amount of clicks:

  • Line - polyline
  • Rectangle - 2 clicks
  • Circle - 2 clicks
  • Ellipse - 2 clicks
  • Arc - 3 clicks
  • Spline - polyline
  • Dimension - 3 clicks
  • Textbox - 1 click
  • Textbox With Leader - 2 clicks
  • Viewport - 1 click (only available in sheets)

Some objects like the Line and Spline objects are polylines, meaning that they wont stop drawing until you press the escape key on your keyboard.
To change for example the line width and stroke color of the object that will be drawn, you can change the drawing properties in the drawing palette.

Selecting objects

To select an object you need to activate the select tool in the tool-bar.
You can select an object by clicking on it or by clicking on a snap point on that object.
You can also drag the mouse (with the left mouse button down) to select the objects inside of the area that appears.
If you press the control key while clicking on an object or dragging the mouse, any object that were previsly select will remain selected. Select an object by pressing on it or by pressing on a snap point on that object. To select multiple object you can drag the mouse around the objects to select them. An object needs to be fully within the reactangle for it to be selected.
You can also select multiple objects by pressing control when pressing on an object.

Tool-bar

The tool-bar which is located on the left side is used to select the current tool and to open the drawing palette.
There are 11 different tools, the first one is used for selecting objects and the other are used to draw objects

Command-bar

The command-bar is similar to the tool-bar in many ways but it is only available when you have one or more objects selected and instead of selecting a tool it allows you to activate a command on the currently selected objects

Drawing palette

The drawing palette lets you change different drawing properties/settings on the next object that will be drawn.
Here is a list of all of the drawing properties:

  • Stroke color - the color of the stroke
  • Line width - the width of the stroke
  • Line dash - the dash of the stroke
  • Font size - the size of the text
  • Fill color - the color inside of an object
  • Fill active - if the fill should be visible
  • Show measurements - whether to show the measurements of an object
  • Scale text by line width - whether to scale the text size by the line width instead of the font size
  • Selected image - the selected image of the image
  • Image active - if the selected image should be visible
  • Insulation active - whether to activate insulation
  • Insulation height - the height of the insulation
  • Insulation radius - the radius of the insulation
  • Visible angle - the amount of visible degrees
  • Angle shift - how many degrees the visible angle should rotate/shift
  • Border radius - the radius of the corners on the object

Not every drawing property is available for every object, when you select a tool you can see which drawing properties that are grayed out and which ones that are available for that object.

Context menu

The context menu is similar to the drawing palette in that it has the almost same drawing propeties, now called object properties with the exception of one extra: "Text"
The context menu is only active when one or more objects are selected

Command box

The command box lets you specify precise distances when drawing and moving objects.
The command box can either take a single value representing a distance or two values representing a distance in the x-axis and y-axis
One cool feature that you can use with the command box is to leave one section empty (a section is basically a measurement). You can for example type ",100cm" or "100m," (note the ",") now you're still using two sections because of the "," but one of the sections is empty, when a section is empty it will default to the mouse position, so you could for example draw a line where the x coordinate is calculated based on the number you provided in the command box and the y coordinate is calculated based on the mouse position.
